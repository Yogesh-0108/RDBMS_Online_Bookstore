# Online Bookstore Database

The Online Bookstore Database is designed to manage books, customers, orders, payments, deliveries, reviews and inventory while handling concurrent transactions efficiently.

This project is implemented using MySQL and demonstrates core database concepts such as DDL, DML, DQL, TCL, Stored Procedures, Triggers, Views and Events.

---

## Prerequisites
- MySQL Server 8.0
- MySQL Workbench 8.0 CE
- MySQL Shell (optional)

## Project Highlights
- Designed a normalized relational database with proper constraints
- Implemented real-time inventory management using triggers
- Used stored procedures to enforce business rules
- Created views, events and reports for analytics and automation
- Handled concurrent transactions safely using TCL.
